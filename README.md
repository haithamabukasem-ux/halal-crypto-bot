# بوت إشارات تداول العملات الرقمية (Halal Screening + Signals Only)

## تنويه أولاً وقبل كل شيء
لا يوجد أي بوت أو استراتيجية تداول بدقة 100%. هذا البرنامج **يرسل إشارات مدروسة فنياً**
عبر تيليجرام فقط، ولا ينفّذ أي صفقة تلقائياً على حسابك. القرار النهائي والمسؤولية عليك دائماً.

كذلك: تصنيف "حلال/حرام" لعملة معينة مبني على اجتهاد ومعايير عامة (تجنب الربا، الميسر،
الغرر المفرط) وليس فتوى. راجع أهل الاختصاص الشرعي، وعدّل القائمة في `config.py` بما تطمئن إليه.

## آلية العمل
1. **الفلترة الشرعية**: البوت يراقب فقط قائمة عملات محددة سلفاً في `config.py`
   (`HALAL_SCREENED_SYMBOLS`)، ويتجاهل أي رمز يحتوي كلمات مثل `UP/DOWN/BULL/BEAR`
   (منتجات رافعة مالية).
2. **لا هامش ولا عقود آجلة**: البوت يعمل فقط على بيانات السوق الفوري (Spot) من Binance
   العامة (لا يحتاج مفتاح API لهذا الجزء)، ولا يقترح بيعاً على المكشوف مطلقاً.
3. **الاستراتيجية** (`strategy.py`):
   - اتجاه عام صاعد على فريم 4 ساعات (EMA21 > EMA50).
   - دخول على فريم الساعة عند خروج RSI من تشبع بيعي + تقاطع MACD إيجابي + قرب السعر
     من دعم EMA50.
   - **هدف أول: 2%** من سعر الدخول.
   - **وقف الخسارة**: ليس نقطة تُفعَّل لحظياً، بل يُفعَّل فقط عند **إغلاق شمعة 4 ساعات**
     تحت أدنى نقطة الشمعة المرجعية (لتفادي الخروج بسبب تذبذب لحظي/فتيل عابر) — تماماً
     كما طلبت.
4. **الإرسال**: عند تحقق شروط الدخول أو تفعيل وقف الخسارة، يُرسل البوت رسالة تفصيلية
   إلى تيليجرام تتضمن السبب الفني الكامل (شفافية كاملة - لا "صندوق أسود").

## التشغيل
```bash
pip install -r requirements.txt
```

عدّل ملف `config.py`:
```python
TELEGRAM_BOT_TOKEN = "التوكن من BotFather"
TELEGRAM_CHAT_ID   = "الآيدي الخاص بك أو بالقناة"
```

للحصول على التوكن: تحدث مع [@BotFather](https://t.me/BotFather) في تيليجرام وأنشئ بوت جديد.
للحصول على chat_id: تحدث مع [@userinfobot](https://t.me/userinfobot).

ثم شغّل:
```bash
python main.py
```

## التطوير القادم (اقتراحات)
- إضافة تأكيد حجم تداول (Volume) لتقوية الإشارة.
- دعم أهداف متعددة (TP2, TP3) بنسب مختلفة.
- تشغيل دائم على سيرفر (VPS) أو كخدمة (systemd / Docker) بدل تشغيله يدوياً.
- لاحقاً: تفعيل التنفيذ التلقائي عبر Binance API (مفتاح API خاص + صلاحيات Spot Trading
  فقط) بعد اختبار الإشارات يدوياً لفترة كافية والتأكد من أدائها.

## تحذير أخير
الأسواق متقلبة وغير مضمونة. لا تضع أموالاً لا تتحمل خسارتها. هذا البرنامج أداة مساعدة
لتنظيم القرار، وليس ضماناً للربح.
